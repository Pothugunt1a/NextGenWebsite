RT NEXTGENAI - CPANEL DEPLOYMENT INSTRUCTIONS
=====================================================

STEP 1: COMPLETE PRODUCTION BUILD
---------------------------------
Before uploading to cPanel, you need to generate the production JavaScript and CSS files:

1. Run this command in your development environment:
   npm run build

2. Copy the generated files from dist/public/ to the cpanel-deploy folder:
   - Copy dist/public/assets/*.js to cpanel-deploy/assets/
   - Copy dist/public/assets/*.css to cpanel-deploy/assets/
   - Update cpanel-deploy/index.html to reference the actual generated filenames

STEP 2: UPLOAD TO CPANEL
------------------------
1. Log into your cPanel File Manager
2. Navigate to public_html directory
3. Upload ALL files from cpanel-deploy folder:
   ✓ index.html
   ✓ .htaccess
   ✓ contact.php
   ✓ generated-icon.png
   ✓ assets/ folder (all images and validation photos)
   ✓ README-DEPLOYMENT.md

STEP 3: VERIFY DEPLOYMENT
-------------------------
1. Visit your domain to test the website
2. Check all pages: Home, Validation, Life Science, Careers, About, Contact
3. Verify all images load correctly
4. Test contact form functionality
5. Ensure mobile responsiveness
